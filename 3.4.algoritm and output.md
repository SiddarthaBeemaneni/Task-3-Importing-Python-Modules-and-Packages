3.4 Count Non-Repeated Digits

Scenario: Write a program to find the count of non-repeated digits in a given number M. The number will be passed as an integer input.

Algorithm:

1. Input a positive integer N from the user.

2. Convert the number to a string to access each digit.

3. Create a dictionary to count occurrences of each digit.

4. Iterate through each digit and increment its count in the dictionary.

5. Initialize a counter for non-repeated digits.

6. Iterate through the dictionary and increment the counter if a digit count is exactly 1.

7. Display the count value.

output:

input         output

292            1

1015           2

108            3

22             0
