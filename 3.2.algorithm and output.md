3.2 Check Perfect Square After Adding 1

Scenario: Given an integer N, check whether N + 1 is a perfect square.

Algorithm:

1. Input an integer N from the user.

2. Calculate N_plus_1 = N + 1.

3. Compute the square root of N_plus_1.

4. If the square root is an integer, then it is a perfect square.

5. Display "Yes" if it is a perfect square, otherwise "No".

output:

input         output

24             yes

26             no

36             yes
